СВОЙСТВА FLEX-КОНТЕЙНЕРОВ

display: flex;

Назначение: Включает флексы

Возможные значения: flex / inline-flex

flex-direction: row;

Назначение: указывает в какую строку элементам лечь

Возможные значения: row / row-reverse / column / column-reverse

flex-wrap: nowrap;

Назначение: указывает можно ли распологать текст в несколько строк

Возможные значения: nowrap / wrap / wrap-reverse

flex-flow: row nowrap;

Назначение: sadhflhfdshajfha

justify-content: flex-start;

Назначение: расположите элементы по основной оси

Возможные значения: flex-start / flex-end / center / space-between / space-around

align-items: stretch;

Назначение: расположите элементы по вертикали

Возможные значения: flex-start / flex-end / center / baseline / stretch

align-content: stretch;

Назначение: выравнивает строки элементов по вертикали

Возможные значения: flex-start / flex-end / center / space-around / stretch / space-between
